头文件-string&cstring

string

  1. 构造函数：
    • string(): 创建一个空字符串。
    • string(const char* s): 用 C 风格字符串初始化字符串。
    • string(const string& str): 复制构造函数。
  2. 赋值和拼接：
    • operator=: 赋值操作符。
    • assign(const char* s): 用 C 风格字符串赋值。
    • assign(const string& str): 复制赋值。
    • +=: 字符串拼接。
  3. 访问和修改字符：
    • operator[]: 访问字符。
    • at(): 访问字符，提供范围检查。
    • front(), back(): 返回第一个和最后一个字符。
    • data(): 返回指向字符数组的指针。
  4. 字符串信息获取：
    • size(), length(): 返回字符串长度。
    • empty(): 判断字符串是否为空。
  5. 字符串比较：
    • compare(const string& str): 字符串比较。
    • ==, !=, <, <=, >, >=: 字符串比较运算符。
  6. 子串操作：
    • substr(pos, len): 返回从位置 pos 开始、长度为 len 的子串。
  7. 查找和替换：
    • find(), rfind(): 查找字符串或字符的位置。
    • replace(): 替换字符串的一部分。
  8. 插入和删除：
    • insert(): 在指定位置插入字符或字符串。
    • erase(): 删除字符或子串。
  9. 转换为 C 风格字符串：
    • c_str(): 返回一个指向以空字符结尾的字符数组的指针。

相关函数：

  1. 输入输出：
    • getline(): 从输入流中读取一行。
    • operator<<, operator>>: 用于输出和输入字符串。
  2. 字符串连接：
    • operator+: 两个字符串连接。
  3. 字符串长度：
    • strlen(): 计算 C 风格字符串的长度，不适用于 std::string。
  4. 字符串比较：
    • strcmp(): 比较两个 C 风格字符串的大小，不适用于 std::string。

cstring

  1. 字符串复制：
    • strcpy(dest, src): 将字符串 src 复制到字符串 dest。
    • strncpy(dest, src, n): 将字符串 src 的前 n 个字符复制到字符串 dest。
  2. 字符串连接：
    • strcat(dest, src): 将字符串 src 连接到字符串 dest 的末尾。
    • strncat(dest, src, n): 将字符串 src 的前 n 个字符连接到字符串 dest 的末尾。
  3. 字符串比较：
    • strcmp(str1, str2): 比较字符串 str1 和 str2。
    • strncmp(str1, str2, n): 比较字符串 str1 和 str2 的前 n 个字符。
  4. 字符串长度：
    • strlen(str): 返回字符串 str 的长度。
  5. 字符串搜索：
    • strchr(str, ch): 在字符串 str 中查找字符 ch 的第一次出现。
    • strrchr(str, ch): 在字符串 str 中查找字符 ch 的最后一次出现。
    • strstr(str, substr): 在字符串 str 中查找子串 substr 的第一次出现。
  6. 字符串分割：
    • strtok(str, delim): 将字符串 str 按照分隔符 delim 进行分割。
  7. 内存操作：
    • memcpy(dest, src, n): 将 src 的前 n 个字节复制到 dest。
    • memmove(dest, src, n): 将 src 的前 n 个字节复制到 dest，处理重叠内存区域。
    • memset(str, value, n): 将字符串 str 的前 n 个字节设置为指定的值。
  8. 字符串转换：
    • atoi(str), atol(str), atof(str): 将字符串转换为整数、长整数和浮点数。
