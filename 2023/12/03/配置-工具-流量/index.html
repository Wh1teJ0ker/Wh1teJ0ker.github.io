配置-工具-流量

记录一下自己用过的流量工具及链接地址

综合分析

名称 链接 描述
Wireshark Wireshark · Download 流量鲨鱼，经典中的经典，无需多言，用就完事了
Brute Shark odedshimon/BruteShark: Network Analysis Tool (github.com) 偶然接触到的一款流量分析工具，好处是能自动生成拓扑图之类的，还有一些集成性的hash爆破功能，感觉还行。暂时用的不算特别多，之后再说更多的体验感受吧

嗅探提取

名称 链接 描述
NTLMRawUnHide mlgualtieri/NTLMRawUnHide NTLMRawUnhide.py 是一个 Python3 脚本，旨在解析网络数据包捕获文件并以可破解的格式提取 NTLMv2 哈希值。该工具旨在从本机 Windows 二进制文件（如 NETSH.EXE 和 PKTMON.EXE）生成的文件中提取 NTLMv2 哈希，而无需转换。（应该是协议问题，之前提取失败过）
net-creds DanMcInerney/net-creds 相比上面一个，更加综合性的嗅探工具（个人推荐）
