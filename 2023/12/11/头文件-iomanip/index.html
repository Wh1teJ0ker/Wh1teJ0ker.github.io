头文件-iomanip

iomanip

简介

主要用于基本的输入输出格式调整

用法

函数/类 用法 描述
setw(int n) cout << setw(5) << 123; 设置域宽为 n，影响后续输出的宽度。
setprecision(int n) cout << setprecision(2) << 3.14159; 设置浮点数的精度为 n。
setfill(char c) cout << setfill('*') << setw(5) << 123; 设置填充字符为 c。
setiosflags(ios_base::fmtflags flags) cout << setiosflags(ios::scientific) << 123.456; 设置流格式标志，比如设置科学计数法。
resetiosflags(ios_base::fmtflags flags) cout << resetiosflags(ios::scientific) << 123.456; 清除流格式标志。
left cout << left << setw(5) << 123; 左对齐输出。
right cout << right << setw(5) << 123; 右对齐输出。
internal cout << internal << setw(5) << -123; 在符号和数值之间对齐，符号在左，数值在右。
showpos cout << showpos << 123; 在正数前显示加号。
scientific cout << scientific << 123.456; 以科学计数法输出。
fixed cout << fixed << 123.456; 以固定小数点格式输出。
hex cout << hex << 255; 以十六进制格式输出。
oct cout << oct << 255; 以八进制格式输出。
